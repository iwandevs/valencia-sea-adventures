# Valencia Sea Adventures

Aplicación educativa interactiva inspirada en la fauna marina, los hábitats acuáticos y la experiencia de visita al Oceanogràfic de València, diseñada para una sesión de informática de **6º de Primaria**.

El objetivo del proyecto es que el alumnado aprenda contenidos relacionados con animales marinos, ecosistemas, conservación del medio ambiente y orientación visual mediante distintas actividades digitales interactivas.

---

## Objetivo del proyecto

Esta aplicación nace como una propuesta educativa para una clase de informática de 6º de Primaria.

La actividad está pensada para tener una duración aproximada de **40 minutos**, permitiendo que cada alumno introduzca su nombre y complete una serie de retos interactivos de forma autónoma o guiada por el docente.

El proyecto combina:

- Aprendizaje sobre el medio marino.
- Competencia digital.
- Comprensión lectora.
- Pensamiento lógico.
- Interacción con contenidos visuales.
- Uso responsable de la tecnología en el aula.

## Tipos de actividades

La aplicación está diseñada para incluir diferentes dinámicas educativas.

### 1. Registro del alumno

El alumno introduce su nombre antes de comenzar la actividad.

Esto permite personalizar la experiencia y mostrar mensajes adaptados durante el recorrido.

### 2. Cuestionarios

Preguntas tipo test sobre animales, zonas, hábitats y curiosidades relacionadas con el mundo marino.

### 3. Verdadero o falso

El alumno debe identificar si determinadas afirmaciones son correctas o incorrectas.

### 4. Une conceptos

Actividad de relación entre animales, hábitats, características o zonas del recinto.

### 5. Busca en el mapa

Actividad visual donde el alumno debe localizar zonas o elementos en un mapa simplificado.

### 6. Reto final

Actividad de repaso general para reforzar los contenidos trabajados.

### 7. Resultado final

Pantalla de cierre con puntuación, resumen y mensaje personalizado.
